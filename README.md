# Todo App - Node.js + Express

A simple and clean Todo List application built with Node.js, Express.js, EJS, and  CSS. 
This was my first full backend project where I learned routing, templating, and serving static files.

## ✨ Features
- Add Task: Add new tasks to your todo list
- Mark Complete/Incomplete: Toggle task status
- Delete Task: Remove tasks from the list
- Responsive UI: Clean and simple design with custom CSS
- Auto Restart: Uses Nodemon for development

## 🛠️ Tech Stack
- Backend: Node.js, Express.js
- Templating Engine: EJS
- Frontend: HTML5, CSS3
- Dev Tool: Nodemon
